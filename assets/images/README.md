# Images Directory

This directory is for storing images used on the website.

## Recommended Images

- **Lab photos**: Group photos, lab workspace, research activities
- **Member photos**: Individual photos for lab members (replace placeholder icons)
- **Research images**: Visualizations, diagrams, or photos related to research projects
- **Logo**: Lab or university logo (if available)

## Image Guidelines

- Use high-quality images (but optimize file sizes for web)
- Recommended formats: JPG, PNG, WebP
- Suggested dimensions:
  - Member photos: 400x400px or larger (square aspect ratio)
  - Hero background: 1920x1080px or larger
  - Research images: 800x600px or larger

## Usage

To use images in the website, update the HTML to reference images in this directory:

```html
<img src="assets/images/your-image.jpg" alt="Description">
```
